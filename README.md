# 🚚 Courier Management System

> ⚡ A high-performance, real-world courier management platform built for speed, efficiency, and seamless user experience.

🔗 **Live Demo**: https://hemtej-courier-management.vercel.app/

📂 **GitHub**: https://github.com/Hemant-gupta2006/Hemtej-Courier-Management

---

## 🧠 Problem Statement

Traditional courier entry systems are:
- slow and repetitive  
- not optimized for bulk data entry  
- inefficient for real-world usage  

👉 This project solves that by providing a **spreadsheet-like interface with instant navigation and smart data handling**.

---

## ✨ Key Features

### ⚡ Ultra-Fast Data Entry
- Excel-like workflow (TAB / ENTER navigation)
- Bulk paste support (CTRL + V)
- Inline smart suggestions for faster typing

### 🚀 Instant Navigation (No Reload UX)
- Keep-alive architecture (no remounting)
- Zero loading flicker when switching pages

### 🔄 Smart Data Synchronization
- Shared in-memory caching
- Add/Edit updates reflect instantly across pages
- Background data refresh

### 📊 Advanced Table System
- Built with TanStack Table
- Dynamic editing and real-time updates
- Optimized rendering for performance

### 🎨 Premium UI/UX
- Glassmorphism design
- Smooth animations using Framer Motion
- Fully responsive (mobile-first)

### 🔐 Authentication
-  Email login via NextAuth

### 📁 Excel Export
- Export courier data to `.xlsx` format

## ⚡ Performance Highlights

- 🧠 Stale-While-Revalidate caching strategy

- 🚫 No unnecessary re-renders  
- 📦 Optimized component architecture

## 🧱 Tech Stack

- **Frontend**: Next.js (App Router), React  
- **Styling**: Tailwind CSS  
- **Database**: Prisma ORM (SQLite / Supabase)  
- **Authentication**: NextAuth.js  
- **Table Engine**: TanStack Table  
- **Animations**: Framer Motion
